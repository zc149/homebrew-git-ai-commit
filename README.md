# homebrew-git-ai-commit

Homebrew tap for [git-ai-commit](https://github.com/zc149/git-ai-commit)

## Installation

```bash
brew tap zc149/git-ai-commit
brew install git-ai-commit
```

## Usage

After installation, use `git-ai-commit` to generate AI-powered commit messages:

```bash
# Stage your changes
git add .

# Generate commit message with AI
git-ai-commit
```

For more details, see the [git-ai-commit README](https://github.com/zc149/git-ai-commit#readme).
